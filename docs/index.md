## 🛠 기술 스택
- **Frontend**: Next.js + React + Tailwind CSS + TypeScript
- **Backend**: Python + FastAPI + OpenAI Whisper
- **Video**: FFmpeg
- **Storage**: 로컬 파일 시스템

## 📂 프로젝트 구조
```
audio-to-voice/
├── frontend/          # Next.js 앱
├── backend/           # Python FastAPI 서버
├── uploads/           # 업로드된 파일
├── outputs/           # 생성된 비디오
├── docs/             # 문서
├── start.sh          # 자동 실행 스크립트
└── README.md         # 프로젝트 설명서
```
